# Pesquisa de Satisfação

Este é um projeto de pesquisa de satisfação desenvolvido com Flask e SQLite.

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/mikedmartins/pesquisa-satisfacao.git


2. Navegue até a pasta do projeto:
   ```bash
   cd pesquisa-satisfacao

3. Crie um ambiente virtual e ative-o:
   ```bash
   python -m venv venv
   source venv/bin/activate  # No Windows use `venv\Scripts\activate`

4. Instale as dependências:
   ```bash
   pip install -r requirements.txt

5. Execute o aplicativo:
   ```bash
   python app.py
   
6. Acesse o aplicativo no navegador:
   ```bash
   http://127.0.0.1:5000


Estrutura do Projeto

app.py: Aplicativo Flask principal.

index.html: Página HTML da pesquisa de satisfação.

requirements.txt: Lista de dependências do projeto.
